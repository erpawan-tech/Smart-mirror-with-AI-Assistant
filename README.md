# Smart AI-Powered Magic Mirror

A smart interactive mirror developed using Raspberry Pi 4 and MagicMirror² with AI voice assistant integration. The system provides real-time information and responds to voice commands, creating an intelligent smart home interface.

## Features

- Real-time Clock and Date
- Weather Updates
- Calendar Integration
- AI Voice Assistant
- Voice Command Recognition
- Auto Boot Functionality
- Smart Dashboard Interface
- Raspberry Pi 4 Based System

## Hardware Used

- Raspberry Pi 4
- Monitor/Display
- Two-Way Mirror
- USB Microphone
- Speaker + Amplifier
- Power Supply

## Software Stack

- Raspberry Pi OS
- MagicMirror²
- Python
- Node.js
- Linux
- Speech Recognition
- AI API Integration

## Working

1. System boots automatically.
2. MagicMirror dashboard launches.
3. User says "Hello Mirror".
4. Voice assistant activates.
5. User asks a question.
6. AI processes the request.
7. Response is delivered through the speaker.

## Project Architecture

User Voice
↓
Microphone
↓
Speech Recognition
↓
AI Processing (API)
↓
Response Generation
↓
Speaker Output
